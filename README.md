Chart.js-guten, a simple helper for Chart.js
============================================

Install
-------

```
$ bower install Chart.js-guten
$ npm install chart.js-guten
```


### Usage

```
<div id="doughnut">
  <canvas width="400" height="400"></canvas>
  <legend></legend>
</div>

#doughnut {
  position: relative;
  width: 550px;
}

$("#doughnut legend").innerHTML = doughnut.generateLegend()
```
